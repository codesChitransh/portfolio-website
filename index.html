Tic tac Toe Game using Python

This project demonstrates my proficiency in basic game development and algorithmic logic. The game is entirely text-based, providing a user-friendly interface where two players take turns inputting their moves. The program utilizes a 2D list to represent the game board and employs functions to print the board, check for a winner, and determine if the board is full, resulting in a tie. The code is organized and readable, emphasizing good coding practices.

Automatic door opener using PIR Sensor

In one of my projects, I designed an automatic door opener system employing a Passive Infrared (PIR) sensor, showcasing my skills in hardware integration and automation. The system is built to enhance convenience and efficiency in various environments, such as homes, offices, or commercial spaces. The PIR sensor, a motion detection device, serves as the core component of the setup.

RFID Based Attendance System

In a recent project, I designed and implemented an RFID-based attendance system, showcasing my expertise in both hardware and software development. The system utilizes Radio-Frequency Identification (RFID) technology to streamline and automate the attendance tracking process. Each individual is provided with a unique RFID card or tag, containing a unique identifier. When an individual enters a designated area, they can simply swipe or tap their RFID card near the reader.
